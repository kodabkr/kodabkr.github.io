Sync UI Assets Instantly to Roblox

A secure bridge between your Figma design tools and Roblox Studio. Authorize once, and upload layouts, images, and fonts straight to your inventory.

Install Roblox Plugin Understand Permissions

Understanding Roblox App Scopes

🔍

assets:read

Enables the UI Designer plugin to view existing assets in your Roblox inventory. This helps prevent uploading duplicates and allows you to select already uploaded assets inside your designer panel.

📤

assets:write

Allows the plugin to upload newly generated user interface images, decals, and visual files directly to your Roblox profile, creating a seamless export experience.

🛡️

OAuth 2.0 Security

Tokens are authenticated directly through Roblox. You have complete control to revoke this application's access at any time through your Roblox account settings panel.

What this integration does with your permissions

✓ What is ALLOWED
• Exporting images, custom icons, and interface files from Figma to your Roblox account using the assets:write API.
• Listing your account's own assets to locate and select existing images using the assets:read API.
• Caching your OAuth access token locally in Roblox Studio's secure plugin storage or Figma secure local storage.
✗ What is FORBIDDEN
• No Account Takeover: Cannot read or modify your Roblox password, security settings, or email.
• No Financial Access: Cannot spend Robux, buy or trade items, or view transaction histories.
• No Game Modification: Cannot alter your places, publish scripts, or affect your game servers.
🔒 Cloudflare Workers Integration

Zero-Retention Ephemeral Architecture

Your security is our absolute priority. To sync assets, the Figma plugin communicates with a secure serverless Cloudflare Worker that processes authorization and upload requests.

1
Token Exchange: The OAuth token is forwarded from the Figma plugin to the Cloudflare Worker purely to sign the Roblox API request.
2
Ephemeral Transit: Assets and credentials exist only in the Worker's active RAM for the few milliseconds it takes to complete the upload to Roblox.
3
No Database Storage: We do not operate a database. Your tokens, designs, and Roblox information are never stored and are completely inaccessible to the developer.
