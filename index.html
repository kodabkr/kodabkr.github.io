Secure Transmission

SECURITY

To guarantee the privacy of your developer credentials, the Figma plugin communicates with a secure serverless Cloudflare Worker. This worker operates as an ephemeral proxy.

The OAuth access token and binary design data are processed purely in-memory (RAM) during transit and uploaded directly to official Roblox APIs. The worker never saves logs, holds records, or uses a database. The developer has absolutely no visibility or access to your credentials or files.

Edge Transit Loop

  • 01. Request dispatched from Figma
  • 02. TLS Encryption secures payload
  • 03. Cloudflare Worker processes in RAM
  • 04. Roblox API receives files
  • 05. Worker memory wiped instantly
